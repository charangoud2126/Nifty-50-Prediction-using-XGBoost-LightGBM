# 📈 Nifty 50 Prediction using XGBoost & LightGBM

## 🚀 Project Overview  
This project predicts **Nifty 50 index trends** using machine learning models like **XGBoost** and **LightGBM**. It uses historical stock data and applies hyperparameter tuning to improve accuracy.

## 🎯 Objectives  
- Predict Nifty 50 price movements  
- Compare XGBoost and LightGBM performance  
- Improve model accuracy using tuning  
- Evaluate using regression metrics  

## 📊 Dataset  
- Source: Kaggle  
- Features:
  - Open, High, Low, Close  
  - Volume  
  - Technical indicators  

## 🧠 Models Used  
- XGBoost Regressor  
- LightGBM Regressor  

## ⚙️ Techniques  
- Data Preprocessing  
- Feature Engineering  
- Train-Test Split  
- Hyperparameter Tuning  
- Model Evaluation  

## 📈 Evaluation Metrics  
- R² Score  
- MAE  
- RMSE  

## 🏆 Results  
- Both models performed well  
- LightGBM was faster  
- Tuning improved accuracy significantly  

## 📌 Key Insights  
- Lag features improve predictions  
- Ensemble models perform better  
- Tuning is crucial  

## 🛠️ Tech Stack  
- Python  
- Pandas, NumPy  
- Scikit-learn  
- XGBoost  
- LightGBM  
- Matplotlib, Seaborn
